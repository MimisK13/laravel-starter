## Quick Start
- ```git clone git@github.com:MimisK13/laravel-starter.git```
- ```cd laravel-starter```
- ```composer install```
- ```cp .env.example .env```
- ```php artisan key:generate```
- Create database
- ```npm install && npm run dev```
- ```php artisan migrate --seed```
- ```php artisan serve```
